# 操作系统实验归档
## 实验环境
系统：`win10`、`win10`下`WSL(ubuntu18.04 64位)`以及`ubuntu16.04(32位)`虚拟机

工具：`bochs`、`nasm`、`gcc/g++`、`bximage`、`sasm（linux下的轻量化汇编IDE）`

考虑到兼容性问题，一切代码都用32位编译，并在64位系统下添加32位支持以运行。
